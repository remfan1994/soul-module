```python
# SoulModule_Demo.ipynb
# -*- coding: utf-8 -*-
"""
Demo Notebook: Soul Module 1.0 Visualization
Author: [Your Name]
Date: 2025-11-27
"""

# --- Imports ---
import numpy as np
import matplotlib.pyplot as plt
from soul_module import SoulModule

# --- Initialize Soul Module ---
dim = 5
time_steps = 100
dt = 0.1

engine = SoulModule(dim=dim, dt=dt, time_steps=time_steps)

# --- Run the module ---
engine.run()

# --- Extract data ---
B_history = engine.B  # Beneficence over time
R_history = engine.R  # Mercy over time
C_history = np.array([engine.C for _ in range(time_steps)])  # Current consciousness (final repeated for demo)
actions_history = np.array(engine.actions_history)

# --- Compute norms for plotting ---
B_norms = np.linalg.norm(B_history, axis=1)
R_norms = np.linalg.norm(R_history, axis=1)
A_norms = np.linalg.norm(actions_history, axis=1)

# --- Plot Beneficence and Mercy ---
plt.figure(figsize=(12, 6))
plt.plot(engine.times, B_norms, label="Beneficence (||B||)", color="green", linewidth=2)
plt.plot(engine.times, R_norms, label="Mercy (||R||)", color="red", linewidth=2)
plt.plot(engine.times, A_norms, label="Action magnitude (||A||)", color="blue", linestyle="--")
plt.title("Soul Module 1.0: Beneficence and Mercy Over Time")
plt.xlabel("Time")
plt.ylabel("Vector Norms")
plt.legend()
plt.grid(True)
plt.show()

# --- Optional: plot moral trajectory in 2D projection ---
plt.figure(figsize=(8, 6))
plt.plot(B_history[:,0], B_history[:,1], color="green", label="Beneficence trajectory (B[0], B[1])")
plt.quiver(B_history[:-1,0], B_history[:-1,1], R_history[:-1,0], R_history[:-1,1], 
           angles='xy', scale_units='xy', scale=0.1, color='red', alpha=0.5, label="Mercy vectors (R)")
plt.title("Soul Module 1.0: Moral Trajectory (2D Projection)")
plt.xlabel("B[0]")
plt.ylabel("B[1]")
plt.legend()
plt.grid(True)
plt.show()
```

### **Notebook Explanation**

1. **Initialization**: Instantiates `SoulModule` with high-dimensional vectors.
2. **Run**: Executes the full reasoning engine, including continuous integration for B, derivative-based R, and mastery principle.
3. **Data extraction**: Grabs beneficence (B), mercy (R), and actions (A) over time.
4. **Norms plot**: Plots (|B|), (|R|), and (|A|) to show cumulative work, instantaneous trajectory, and action magnitude.
5. **Trajectory plot**: Projects the first two dimensions of B and R to visualize moral path and local direction vectors.
